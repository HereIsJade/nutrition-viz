# Environment setup

1. To set up the server:

    ```
    cd nutrition-backend
    npm install
    node node index.js
    ```

    Now you can try out the playground tool at http://localhost:4000/

2. For the frontend:

    ```
    cd nutrition-graph
    npm install
    npm start
    ```

    Visualizaion can ve viewed at http://localhost:3000/
